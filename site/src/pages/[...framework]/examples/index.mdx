## Framer Motion

NumberFlow was designed to integrate well with [Framer Motion layout animations](https://www.framer.com/motion/layout-animations/):

<FramerMotionLayoutAnimations client:visible>
<Code code={FramerMotionLayoutAnimationsCode} lang="tsx" />
</FramerMotionLayoutAnimations>

**Note:** whenever `NumberFlow` is used within a layout animation it must be (or be in) a
[`motion` component](https://www.framer.com/motion/component/#custom-components) with the
[`layout layoutRoot`](https://www.framer.com/motion/layout-animations/###item-is-performing-unwanted-layout-animations-as-surrounding-content-changes)
props.

To synchronize NumberFlow's transitions with Framer Motion's layout animations,
the package exports a `useDefaultTransformTransition` hook from `@number-flow/react/framer-motion` that
returns a Framer transition to match NumberFlow's default `transformTiming`.
